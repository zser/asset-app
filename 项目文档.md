# 家庭资产管理 App - 项目交接文档

> 供新对话快速了解项目全貌，直接上手开发。

---

## 一、项目概览

| 项目 | 说明 |
|------|------|
| 类型 | 移动端 PWA 应用 |
| 文件 | `index.html` + `manifest.json` + `sw.js` + 图标 |
| 技术栈 | HTML + Tailwind CSS (CDN) + Chart.js + Font Awesome + PWA |
| 运行方式 | GitHub Pages 部署，浏览器/添加到主屏幕运行 |
| 目标用户 | 家庭资产共同管理（夫妻二人） |
| 访问地址 | `https://<用户名>.github.io/<仓库名>/index.html` |

## 二、页面结构（共 8 页）

### 1. 首页（page-home）
- **总资产卡片**：蓝色渐变背景，显示总资产金额
  - 右上角刷新按钮（带旋转动画），点击更新"最后更新时间"
  - 右下角"编辑"按钮 → 进入编辑流程
  - 4个分类占比：股票/基金/活期/存款，各显示金额和百分比
- **Tab 切换**：股票 / 基金 / 活期 / 存款（4 个 Tab）
  - 每个 Tab 下列出该分类的所有数据源
  - 每项显示：图标 + 名称 + 占比 + 金额 + 删除按钮
  - 底部有"添加"按钮（弹窗输入名称和金额）

### 2. 编辑-选择数据源（page-edit-source）
- 标题栏："编辑"，返回按钮回首页
- **分类切换**：顶部下拉选择（活期理财/股票/基金/存款），默认选中"活期理财"
  - 蓝色高亮当前分类，其他灰色
- **数据源列表**：
  - 每项：圆形选择框 + 图标 + 名称 + 金额 + 红色减号删除按钮
  - 选中项蓝色背景，选择框蓝色填充
  - 底部"添加新数据源"（虚线 + 号），选中后进入输入页
- **确定按钮** → 进入数字键盘输入页

### 3. 编辑-输入数据源（page-edit-amount）
- 标题栏："输入数据源"，返回按钮回编辑页
- 顶部显示选中的数据源名称
- 中间显示输入金额 + "元"单位
- **数字键盘**：1-9 / 0 / 00 / . / C(清除) / 退格 / 确认(✓) / 取消(×)
  - 小数点限制2位
  - 确认 → 保存并返回首页（自动更新总资产和占比）
  - 取消 → 返回编辑页

### 4. 日历（page-calendar）
- 月份导航（左右箭头切换）
- **类型筛选**：总金额 / 股票 / 基金（总金额 = 股票 + 基金，默认总金额）
- 7×N 日历网格，每个日期格：
  - 有数据：正数红色背景/文字，负数绿色背景/文字
  - 今天：蓝色背景白色文字
- **点击格子弹窗**：输入股票收益和基金收益，保存后自动计算总额

### 5. 图表（page-chart）
- **资产分布饼图**（Doughnut）：红=股票，橙=基金，蓝=活期，紫=存款
- **月度收益折线图**（Line）：点颜色反映正负（红涨绿跌）
- ⚠️ 用户明确要求"图表一点都别改"

### 6. 心愿首页（page-wish）
- 进行中心愿列表（渐变蓝色卡片）
  - 显示：名称、日期、进度条、已存/目标金额/百分比
  - 点击进入详情
- 历史心愿列表（灰色，点击弹出完成信息）
- 右上角"+"按钮 → 添加心愿页

### 7. 心愿详情（page-wish-detail）
- 进度条 + 已存/目标金额 + 存入按钮
- **达成后心愿区域**（黄色卡片）：展示每个参与人的达成后心愿
- **计划卡片**（按参与人分组）：
  - 哥哥=橙色边框，妹妹=绿色边框
  - 每个卡片内显示该参与人的达成后心愿
  - 每条任务有 checkbox，点击切换完成状态（灰+删除线 / 正常）
  - "添加任务"输入框 + 按钮
  - 卡片右上角可删除整个计划
- 底部"添加计划"按钮（弹窗选择参与人 + 输入达成后心愿）
- **存入达目标时**：自动弹出庆祝弹窗 + 撒花动效 → 自动转入历史心愿

### 8. 添加心愿（page-add-wish）
- 输入：心愿名称、目标金额
- **添加日期**：Vant 风格滚轮日期选择器（年/月/日三列滑动）
- **完成日期**：同上，支持"随缘"选项
- 参与人勾选（哥哥/妹妹），每人可填写达成后心愿
- 支持只勾选一个人（单人心愿）
- 保存后跳转心愿首页

## 三、数据结构

```javascript
// 资产数据
assetData = {
    stock: [  { name, amount, icon, color }  ],      // 股票
    fund: [   { name, amount, icon, isAccount, color } ], // 基金
    current: [{ name, amount, isAlipay?, isWechat?, icon, color } ], // 活期
    deposit: [{ name, amount, bank, icon, color } ]   // 存款
};

// 收益数据
profitData = {
    'YYYY-MM-DD': { stock: Number, fund: Number, total: Number }
    // total = stock + fund
};

// 心愿数据
wishes = [{
    name: String,
    current: Number,        // 已存金额
    target: Number,         // 目标金额
    date: String,           // 添加日期
    completeDate: String,   // 目标日期 或 "随缘"
    plans: [{
        person: String,     // "哥哥" 或 "妹妹"
        color: String,      // "orange" 或 "green"
        reward: String,     // 达成后心愿
        tasks: [{
            text: String,
            done: Boolean
        }]
    }]
}];

// 历史心愿（保留完整数据）
wishHistory = [{
    name: String,
    target: Number,
    current: Number,        // 实际存入金额
    date: String,           // 添加日期
    completeDate: String,   // 目标日期
    plans: Array,           // 保留参与人、达成后心愿、任务等
    completedDate: String   // 完成日期
}];
```

## 四、关键交互规则

| 规则 | 说明 |
|------|------|
| **中国股市配色** | 涨=红色(#ef4444)，跌=绿色(#22c55e)，**绝对不能改成国际配色** |

| **图表** | 用户明确要求不改，修改前必须确认 |

## 九、数据同步机制

### 同步方式
通过 GitHub 私有仓库的 `data.json` 文件实现两人共享同步。

### 配置步骤
1. 创建 GitHub 私有仓库（如 `asset-data`）
2. 生成 Personal Access Token（勾选 `repo` 权限）
3. 首页齿轮图标 → 设置页 → 填写 Token / Owner / Repo → 测试连接

### 同步流程
- **页面加载**：先读 localStorage 本地缓存（快速渲染），再从 GitHub 拉取最新数据覆盖
- **数据变更**：立即存 localStorage + 后台推送 GitHub
- **冲突处理**：409 冲突时自动重新拉取 SHA 后重试
- **离线可用**：网络不通时用本地缓存，状态显示"离线"

### localStorage 键名

| 键 | 内容 |
|----|------|
| `asset-app-data` | 完整数据 JSON（本地缓存） |
| `gh-token` | GitHub Token |
| `gh-owner` | 仓库 Owner |
| `gh-repo` | 仓库名 |
| `gh-sha` | 当前文件 SHA |
| `last-sync-time` | 上次同步成功时间 |
| `hide-amount` | 是否隐藏总资产金额 |

## 十二、部署到 GitHub Pages

### 步骤 1：创建仓库

1. 在 GitHub 创建新仓库（如 `asset-app`）
2. 将以下文件上传到仓库根目录：
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`

### 步骤 2：开启 GitHub Pages

1. 进入仓库 Settings → Pages
2. Source 选择 `Deploy from a branch`
3. Branch 选择 `main`（或你的分支），目录选 `/ (root)`
4. 点击 Save

### 步骤 3：访问应用

等待部署完成后，访问：
```
https://<你的用户名>.github.io/<仓库名>/index.html
```

### 步骤 4：添加到主屏幕（手机桌面图标）

**iOS (Safari)**：
1. 打开网页
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

**Android (Chrome)**：
1. 打开网页
2. 点击右上角菜单
3. 选择"添加到主屏幕"或"安装应用"

### 步骤 5：配置 GitHub 同步

首次打开应用后：
1. 点击首页总资产卡片右上角齿轮图标
2. 输入 GitHub Token / Owner / Repo（数据仓库，可以和应用仓库不同）
3. 点击"测试连接"
4. 配置成功后，两台手机访问同一地址，数据自动同步
